# Estudos-Programação
Repositório com meus estudos, práticas e projetos desenvolvidos durante minha jornada no mundo da programação.

# 🚀 Objetivo
Desenvolver lógica de programação e consolidar os fundamentos da linguagem Java através de exercícios práticos.

# 📑​ Conteúdo
* **Fundamentos**: primeiros programas em Java, variáveis e tipos primitivos.
* **Operadores**: cálculos e manipulação de valores com operadores aritméticos, relacionais e lógicos.
* **Controle de Fluxo**: estruturas condicionais como `if/else` e `switch-case`.
* **Loops**: estruturas de repetição como `for`, `while` e `do-while`.
* **POO (Programação Orientada a Objetos)**: classes, objetos, atributos, métodos e encapsulamento.
* 
# 🛠 Tecnologias
* Java ☕
* Spring Tool Suite (STS) – IDE baseada no Eclipse
* Sublime Text (editor de código)
* PowerShell (terminal para compilação e execução)
